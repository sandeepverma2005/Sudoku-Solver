#Sudoku Solver 
a powerful c++ program that solves sudoku puzzles using the "BACKTRACKING ALGORITHM".

#Features
*solve any valid 9x9 Sudoku board.
*efficient and fast execution.

#Built With
1: language-> c++;
2:  logic-> Recursive Backtracking ..

#How to run
1: Download the "sudoku_solver.cpp" file.
2. compile it using a c++ compiler: 
         g++ sudoku_solver.cpp-o solver
